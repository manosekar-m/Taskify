**Taskify** is a premium, high-performance task management and productivity application built with Flutter. It combines a sleek, modern aesthetic with powerful scheduling tools to help you stay organized, focused, and efficient throughout your day.

---

## ✨ Key Features

### 📅 Smart Scheduling
*   **Intuitive Task Creation**: Quickly add tasks with precise start/end times and dates.
*   **Dynamic Dashboard**: View your daily schedule with a real-time "Global (UTC)" and "Local Time" clock.
*   **Calendar Integration**: Plan ahead or review past achievements with a fully integrated calendar view.

### ⏱️ Productivity Tools
*   **Focus Mode**: A dedicated Pomodoro-style countdown timer for every task. Stay in the zone with visual progress indicators and audio alerts.
*   **Rough Notes**: A built-in scratchpad for quick jottings and ideas that don't need a formal schedule. Featuring full CRUD (Add, Edit, Delete) support and local persistence.

### 🔔 Advanced Notifications
*   **Early Alert System**: Set custom "Early Alert Delays" (e.g., 5, 10, or 30 minutes before) to prepare in advance.
*   **Reliable Reminders**: Automated notification management that ensures you never miss a deadline.

### 🎨 Premium UI/UX
*   **Dynamic Themes**: Seamless support for stunning Dark and Light modes.
*   **Modern Aesthetic**: Glassmorphism effects, smooth gradients, and Material 3-inspired components.
*   **Fluid Gestures**: Swipe-to-complete and swipe-to-delete functionality for effortless management.
*   **High-End Pickers**: Custom-themed time and date pickers with smooth, rounded interfaces.

---

## 🛠️ Tech Stack

-   **Frontend**: Flutter & Dart
-   **Database**: Hive (NoSQL local storage)
-   **Notifications**: Flutter Local Notifications
-   **Utilities**: Intl (Date/Time formatting), Flutter Launcher Icons, Flutter Native Splash

## 🚀 Getting Started

### Prerequisites
-   [Flutter SDK](https://docs.flutter.dev/get-started/install)
-   Android Studio / Xcode
-   Dart SDK

### Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/manosekar-m/Taskify.git
    ```

2.  **Navigate to Project Directory**:
    ```bash
    cd Taskify
    ```

3.  **Install Dependencies**:
    ```bash
    flutter pub get
    ```

4.  **Run the App**:
    ```bash
    flutter run
    ```

---

## 📖 How to Use
1.  **Splash & Login**: Start by creating an account to keep your local data organized.
2.  **Add Tasks**: Tap the floating **+** button to create your schedule.
3.  **Focus**: Tap **Focus** on any task card to start a deep-work timer.
4.  **Notes**: Use the **Rough Notes** preview on the home screen to access your quick-jotting area.
5.  **Settings**: Visit the **Profile** screen to toggle themes, notification settings, and view the "How to Use" guide.
